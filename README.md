\# Ecommerce Sales Analysis Dashboard



📄 Detailed Report: \[View Project Report](docs/Ecommerce\_Dashboard\_Report.pdf)



📊 Multi-sheet Excel analysis covering trends, segments, geography, and performance metrics.



\---



\## Project Overview

The Ecommerce Sales Analysis Dashboard is an Excel-based data analytics project designed to analyze ecommerce performance across multiple dimensions such as sales trends, customer segments, geographic distribution, and product performance.



Instead of a single dashboard, this project uses multiple analytical sheets to break down different aspects of the business and provide detailed insights.



\---



\## Objectives

\- Analyze overall sales performance

\- Identify top-performing products/customers

\- Understand customer segmentation

\- Explore geographic sales distribution

\- Study sales trends over time



\---



\## Tools Used

\- Microsoft Excel

\- Pivot Tables

\- Pivot Charts

\- Data Cleaning

\- Analytical Reporting



\---



\## Dataset

The dataset includes ecommerce transaction data such as:

\- Order Date

\- Sales

\- Quantity

\- Customer Segment

\- Country / Market

\- Product Category



\---



\## Analysis Breakdown (Sheets)



\- \*\*Dashboard Overview\*\* – High-level summary of metrics  

\- \*\*KPI Cards\*\* – Key performance indicators  

\- \*\*Country Analysis\*\* – Sales distribution by country  

\- \*\*Customer Segment Analysis\*\* – Segment-wise performance  

\- \*\*Market Distribution\*\* – Market contribution analysis  

\- \*\*Trend Analysis\*\* – Sales trends over time
